This wiki supports fully interactive Desmos graphing calculators embedded directly into any article. Graphs are live: you can pan, zoom, and even edit the expressions if the calculator is not locked.

Method 1: HTML Data Attributes (Declarative)

Add a .wiki-desmos div with a data-exprs attribute containing a JSON array of expressions. The wiki.js engine will automatically instantiate the calculator on page load.

Example: $y = \sin(x)$ and its derivative $y = \cos(x)$:

The HTML for this embed is simply:

<div class="wiki-desmos"
     data-exprs='[
       {"latex":"y=\\\\sin(x)","color":"#c74440"},
       {"latex":"y=\\\\cos(x)","color":"#2d70b3"}
     ]'
     style="height:400px;">
</div>

Method 2: JavaScript API (Programmatic)

For more control, call wikiDesmos() from a script block. This gives you full access to the Desmos API.

Example: A family of parabolas $y = ax^2$ with a slider for $a$:

<div id="desmos-parabola" style="height:420px;"></div>
<script>
document.addEventListener('DOMContentLoaded', () => {
  const calc = wikiDesmos('desmos-parabola', [], { keypad: false });
  if (calc) {
    calc.setExpression({ id: 'a', latex: 'a=1', sliderBounds: { min: -3, max: 3 } });
    calc.setExpression({ id: 'p', latex: 'y = a x^2', color: Desmos.Colors.RED });
  }
});
</script>

Method 3: Desmos iframe (No JS Required)

If you prefer zero JavaScript, you can embed a saved Desmos graph as an <iframe> using a public Desmos share link. This works even without the API key.

Desmos Graph — Saved Share Link Example

To get a share link: create your graph on desmos.com, click Share, copy the URL, and append ?embed to it.
