Authoring Articles

This guide covers the conventions and configuration required to author well-formed articles in Mini Wiki.

The WIKI_PAGE Object

Every page should include a window.WIKI_ROOT = '../'; window.WIKI_PAGE script block before the wiki.js script tag. This object provides metadata used by the engine for titles, categories, and future tooling.

window.WIKI_PAGE = {
  title:     'Article Title',       // Used in <title> tag
  category:  'Category Name',       // Displayed in page meta
  tags:      ['keyword', 'topic'],  // For search and filtering
  lastEdited: '2024-06-01',         // ISO date string
  author:    'Author Name',
};

The WIKI_SIDEBAR Object

Each page defines its own sidebar navigation. This allows contextually relevant navigation for every article.

window.WIKI_SIDEBAR = [
  {
    label: 'Navigation',
    links: [
      { text: 'Home',           href: '../../index.html' },
      { text: 'All Categories', href: '../categories.html' },
    ]
  },
  {
    label: 'This Section',
    links: [
      { text: 'This Article',   href: 'this-article.html' },
      { text: 'Related Page',   href: 'related.html' },
    ]
  }
];

Note that href values are relative to the location of the current page, not the site root.

The Search Index

The search index lives in index.html as window.WIKI_SEARCH_PAGES. Every published article should have an entry:

window.WIKI_SEARCH_PAGES = [
  {
    title: 'Article Title',
    url:   'pages/section/article.html',  // Relative to index.html
    desc:  'One sentence description.',
    tags:  ['keyword', 'topic', 'section']
  },
  // ...
];

Breadcrumb Convention

Every article should begin with a breadcrumb trail:

<div class="wiki-breadcrumb">
  <a href="../../index.html">Home</a>
  <span class="sep">&rsaquo;</span>
  <a href="../categories.html">Category</a>
  <span class="sep">&rsaquo;</span>
  <span>Article Title</span>
</div>

Floating TOC

Place this empty div near the top of the content area. The engine will auto-populate it from all h2, h3, and h4 headings on the page.

<div class="wiki-toc floating"></div>

The TOC only renders if the page has three or more headings. It is collapsible and will also populate the right sidebar column.

Writing Style

Mini Wiki articles follow an encyclopedic tone with a clear editorial perspective favoring Western civilization, classical learning, the liberal arts, and Christian ethics. Specific conventions:

  • Write in third person; avoid first-person narration.
  • Use the present tense for statements of fact; past tense for historical events.
  • Bold the subject of the article on its first mention in the introduction.
  • Use hatnotes for disambiguation and "see also" references at the top of articles.
  • Cite sources using inline citation links to a references section at the bottom.
  • Avoid informal language, slang, and speculative claims without citation.

Page Meta Block

Include a page meta block at the end of each article, before the closing </main> tag:

<div class="wiki-page-meta">
  <span>Last edited: <time datetime="2024-01-01">January 1, 2024</time></span>
  <span>Author: Name</span>
  <span>Category: <a href="categories.html">Category</a></span>
</div>
