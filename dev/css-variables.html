CSS Variables

All design tokens are defined as CSS custom properties on :root in wiki.css. Override any variable on a per-page basis or globally.

Theming Example

To override the accent color site-wide, add this to the top of a custom CSS file loaded after wiki.css:

:root {
  --accent:       #1a4c8f;  /* blue instead of crimson */
  --accent-hover: #0f2d5a;
  --accent-bg:    #edf4fc;
}
