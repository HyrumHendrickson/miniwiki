About MyWiki

MyWiki is a personal static wiki hosted on GitHub Pages. It is built with plain HTML, CSS, and JavaScript ‚Äî no server, database, or build step required.

Technology

  • Layout & Components: Custom CSS (assets/css/wiki.css)
  • Interactivity: Vanilla JavaScript (assets/js/wiki.js)
  • Mathematics: KaTeX
  • Graphing: Desmos API
  • Hosting: GitHub Pages

Contributing

To add an article: copy pages/article-template.html, fill in your content, add it to the sidebar and search index in index.html, and submit a pull request.
