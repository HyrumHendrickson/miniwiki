Welcome to MyWiki

This is a self-hosted wiki built with plain HTML, CSS, and JavaScript. It renders LaTeX mathematics, supports interactive Desmos graphs, and includes a wide variety of modular components.

Getting Started

To add a new article, copy pages/article-template.html, fill in your content, and link to it from the sidebar. Register it in the search index by adding an entry to window.WIKI_SEARCH_PAGES in assets/js/wiki.js.

â„¹ï¸ Hosting on GitHub Pages

Push the entire wiki/ folder to a GitHub repository and enable GitHub Pages in the repo settings. Your wiki will be live at https://yourusername.github.io/repo-name/.
