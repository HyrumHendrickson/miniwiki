Component Reference

This page demonstrates every modular component available in MyWiki. All components are driven purely by CSS classes and a small amount of JavaScript in wiki.js.

Infoboxes

Infoboxes float to the right by default. Add class left to float left instead.

Standard Infobox
TypeStandard right-floating
Class.wiki-infobox
FloatRight (default)
Width280px

Place <div class="wiki-infobox"> before or after any paragraph. The box floats to the right and text wraps around it. Use <div class="clearfix"></div> after the wrapped content to reset the float.

Code Blocks

Use <pre><code> for multi-line code blocks. Inline code uses the <code> tag.

// JavaScript example
function fibonacci(n) {
  if (n <= 1) return n;
  return fibonacci(n - 1) + fibonacci(n - 2);
}

console.log(fibonacci(10)); // 55

Figures

Float images left or right with captions using .wiki-figure:

Text wraps around the floating figure. The figure gets a border, border-radius, and caption area automatically. Use .wiki-figure.right for right-floating, or .wiki-figure alone for centered/full-width.

Click any figure or gallery image to open a lightbox overlay. Press Escape or click outside the image to close.
